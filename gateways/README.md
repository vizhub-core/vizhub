## Gateways

This package defines the "gateways" of VizHub.

Corresponds to the "gateways" concept from [Clean Architecture](https://blog.cleancoder.com/uncle-bob/2012/08/13/the-clean-architecture.html).

### Development

Unit tests:

```
npm test
```

Type checking:

```
npm run tsc
```

Or, in watch mode:

```
npm run tsc-watch
```
